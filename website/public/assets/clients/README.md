# Client screenshots

Drop one image per client here, then update each client MDX page to set
`src="/assets/clients/<id>.png"` on its `<Figure>` tags.

Suggested filenames:

- `claude-desktop.png` - config file open in an editor + the plug icon
  in Claude Desktop's conversation header.
- `claude-code.png` - terminal showing `claude mcp add pinax -- pinax
  serve` followed by `claude mcp list`.
- `cursor.png` - Cursor settings panel with the MCP servers list.
- `windsurf.png` - Windsurf Cascade panel listing the Pinax MCP tools.
- `cline.png` - Cline MCP servers panel inside VS Code.
- `copilot.png` - `.vscode/mcp.json` open in VS Code + the MCP: List
  Servers Command Palette entry.

PNG or WebP both work. Aim for around 1600 px wide so the figures stay
sharp on retina displays without bloating the page weight.
